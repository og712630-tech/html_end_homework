Git

  • git status – показывает состояние рабочего каталога и индекса (staging area). Какие файлы изменены, добавлены или готовы к коммиту.
  • git add – добавляет изменения в индекс для следующего коммита. Можно добавить конкретный файл или все изменения (git add .)
  • git commit – сохраняет изменения в репозитории с комментарием. Обязательно указывается сообщение: git commit -m "текст"
